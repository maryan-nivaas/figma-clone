## Overview

## Setup

### Prerequisites
- Node.js (Latest LTS recommended)
- PostgreSQL
- Liveblocks Account

### Installation

1.  **Clone the project**
    ```bash
    git clone <your-repo-url>
    cd figma-clone
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Environment Setup**
    Create a `.env` file in the root directory with the necessary keys (Database URL, Auth secrets, Liveblocks keys).

4.  **Run Development Server**
    ```bash
    npm run dev
    ```

5.  **Access Application**
    Open [http://localhost:3000](http://localhost:3000)
